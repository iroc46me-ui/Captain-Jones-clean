# Vercel + GoDaddy Deployment Steps

1. Create/sign into a Vercel account.
2. Create/sign into GitHub.
3. Upload this project folder to a new GitHub repository.
4. In Vercel, choose **Add New Project** and import the GitHub repository.
5. Use the default Next.js settings and deploy.
6. In Vercel, add the custom domain: `daveyjonesjunkntreasure.com`.
7. Vercel will show DNS records to add in GoDaddy.
8. In GoDaddy DNS, add/update the records Vercel provides.
9. Wait for DNS propagation.
10. Confirm HTTPS/SSL is active in Vercel.

Recommended first launch: Harbor Alpha landing page only, with seller interest collection before full payments or marketplace accounts.
