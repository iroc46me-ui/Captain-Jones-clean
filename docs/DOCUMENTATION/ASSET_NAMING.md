# Asset Naming Standards

Use lowercase names, hyphens or underscores, and clear version numbers.

Examples:
- djjt-primary-logo-v1.png
- djjt-emblem-gold-v1.svg
- harbor-hero-cinematic-v1.webp
- captain-bust-profile-v1.png
- treasure-chest-glow-v1.webp
- social-preview-harbor-alpha-v1.png
- trusted-captain-badge-v1.svg
